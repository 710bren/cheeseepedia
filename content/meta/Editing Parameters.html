+++ title = "Editing Parameters" date = 2024-09-24 draft = false tags = ["Meta"] categories = [] contributors = ["The 64th Gamer"] +++ Parameters are the lines above the main body of an article that dictate where the article goes on the site, how it is formatted, and what prominent info is displayed. The parameters for this page are: 

+++                                 
title = "Editing Parameters"        
date = 2024-09-24                   
draft = false                       
tags = ["Meta"]                     
categories = []                     

                           
contributors = ["The 64th Gamer"]   
+++
Parameters always start and end with +++. Each parameter is also case sensitive. The wiki has a file containing all the parameters available here. If you ever compile the site locally, this is automatically generated with every new article you make. This file also contains comments starting with #, which guides you through each one- and can be erased once you're done using it.

It is extremely important to format parameters the way the wiki expects you to, or it will prevent the site from working.
  • Parameters like title = "" require the text to be encased in those quotes.
  • Parameters like tags = ["Meta","User"] which are for multiple inputs, require each item to be in quotes and to be seperated by commas, also contained in the brackets.
  • The draft = false parameter must always be there and always be false. Turning it true hides it from showing on the site.
  • Any parameter requiring a date must be formatted like YEAR-MONTH-DAY, in other words 2024-12-31
