+++ title = "SPP April 1990 Show" draft = false tags = ["Cyberstar Segments"] date = "2023-09-16" [Article] startDate = "1990-4-00" endDate = "" pageThumbnailFile = "Werfghjkjhgfds.avif" contributors = ["The 64th Gamer","DerekCat"] +++ SPP April 1990 Show is one of the last few showtapes released for Showbiz Pizza Place's hosting the Rock-afire Explosion. For some stores, this was the last tape used before Concept Unification would start at their location. Each tape sent after [SPP Christmas '89 / Wave Bday]({{< ref "wiki/SPP Christmas '89 Wave Bday.html" >}}) would become increasingly more rare as stores would convert to 3-Stages.
