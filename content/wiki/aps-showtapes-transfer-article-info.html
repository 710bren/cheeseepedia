Cheese showtapes. Each file represents a single segment of the tape, or a single live show. Versions of APS are used to this day, though Studio C HD shows have their lights performed in VenueMagic.

Discovery

The current list of APS files is from a pack of 5 floppy cases from CEC Corporate, each containing a selection of APS showtapes. More info on these cases can be found on the {{< wiki-link "1990-1992 APS Floppy Cases" >}} page.
