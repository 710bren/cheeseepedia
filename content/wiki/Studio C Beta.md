The **Studio C Beta** is a stage format found at [Chuck E. Cheese's]({{< ref "wiki/Chuck E Cheese's.md" >}}) locations which either opened with the stage or replaced from 2000 to 2012. It is a downgraded version of the Studio C Alpha, and is the 3rd stage to have the 'Studio C' name after the Studio C Alpha / Beta Mix. There is a 32m version, and a 16m version, with the 16m being more common. they are the most common stage after the Dance Floor, with 100 Studio C Betas left as of September 2022.
