##  Grey Box ## 
The Grey Box is a animatronic controller which contained parts from the Pianocorder with some modification from Creative Engineering. It as suggested uses Pianocorder Biphase as a input to then control a Rock Afire Explosion.

###  I/O ### 
2x Data input (RCA Jack I presume)

Character output connector (Centronics Connector)

###  Hardware Breakdown ### 
3x LM324 QUAD OP-AMP

LM339 QUAD COMP.
