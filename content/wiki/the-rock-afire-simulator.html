+++ title = "The Rock-afire Simulator" contributors = ["BattleXGamer"] citations = ["Official Statement from @BattleXGamer2"] tags = ["Simulators"] categories = ["Creative Engineering"] +++ The Rock-afire Simulator is a program developed in Unreal Engine designed to allow users to both view The Rock-afire Explosion's original showtapes and program new showtapes for The Rock-afire Explosion. The simulator is currently in a work in progress state and is expected to release sometime in 2025 or 2026. It is being worked on in collaboration with Creative Engineering Inc to attempt to accurately represent The Rock-afire Explosion digitally.

Overview

Features

The Simulator encorporates serveral features that users might find interesting or valuable.
  • Watch the entire library of The Rock-afire Explosions showtapes
  • Program songs on The Rock-afire Explosion
  • Export your programmed showtapes to {{< wiki-link "CEI .wav Files" >}} so you can watch them on a Real Show
