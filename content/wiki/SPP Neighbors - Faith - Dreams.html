+++ title = "SPP Neighbors - Faith - Dreams" draft = false tags = ["Cyberstar Segments"] date = "2024-01-19" [Article] startDate = "1990-00-00" endDate = "" pageThumbnailFile = "Neighbors.avif" contributors = ["DerekCat"] +++ SPP Neighbors - Faith - Dreams is one of the last few showtapes released for Showbiz Pizza Place's hosting the Rock-afire Explosion. For some stores, this was the last tape used before Concept Unification would start at their location. Each tape sent after [SPP Christmas '89 / Wave Bday]({{< ref "wiki/SPP Christmas '89 Wave Bday.html" >}}) would become increasingly more rare as stores would convert to 3-Stages. Note: It not comfirm to when this show was released but it was sometime between January or February 1990.
